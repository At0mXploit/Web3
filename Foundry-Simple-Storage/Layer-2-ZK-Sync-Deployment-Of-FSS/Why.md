# Why?

## Layer 2 (L2)

* Network built **on top of Ethereum**
* Makes transactions **faster and cheaper**
* L1 (Ethereum) = secure but slow/expensive
* L2 = fast, low-fee, still inherits Ethereum security

## zkSync

* A **Layer 2 solution** using **zero-knowledge proofs**
* Batches transactions to reduce gas fees
* Keeps Ethereum-level security
* Used for DeFi, NFTs, and testing contracts
